# Sorting / PatikaDev Project

## [16,21,11,8,12,22] -- Merge Sort

#### 1. Stages for Array
```
-   [16,21,11]             [8,12,22]
-   [16,21] [11]           [8,12]  [22]
-   [16] [21] [11]         [8] [12] [22]
-   [16,21] [11]           [8,12] [22]
-   [11,16,21]             [8,12,22]
-   [8,11,12,16,21,22]
```
#### 2. Big-O notation

```
-   T(n) = 2T(n/2) + θ(n)
-   θ(nLogn)
    O(n log n)
```
